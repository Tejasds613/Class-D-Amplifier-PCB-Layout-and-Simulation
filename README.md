# Class-D-Amplifier-PCB-Layout-and-Simulation
## Overview 
This Project present the design and simulation of a discrete Class D amplifier using Kicad and Multisim software. The complete design flow including block-level explanation, component selection, and simulation analysis is documented in the attached report

## Tolls used
PCB designed using KiCad software
Simulation is done in Multisim software
